# alpha-compass sidecar

Stateless Python (FastAPI) service that fetches external data and runs AI
summarization for alpha-compass. It owns no persistent state — DuckDB / SQLite
are owned exclusively by the Rust (Tauri Core) process, which calls this sidecar
over `127.0.0.1` HTTP.

## Security

- Bound to `127.0.0.1` only (never exposed externally).
- Every request must carry `Authorization: Bearer <token>`, where the token is a
  random session token generated by Rust at startup and passed via the
  `ALPHA_COMPASS_SIDECAR_TOKEN` environment variable. Mismatches return `401`.

## Run (dev)

Normally the Rust process spawns this automatically. To run standalone for
debugging:

```sh
# from sidecar/
ALPHA_COMPASS_SIDECAR_TOKEN=devtoken \
  uv run uvicorn app.main:app --host 127.0.0.1 --port 8765
# then:
curl -H "Authorization: Bearer devtoken" http://127.0.0.1:8765/health
```

## Layout (per architecture doc §13)

```
app/
├── main.py        # FastAPI app, health, auth middleware
├── adapters/      # one module per data source (later sessions)
├── normalize/     # raw -> common schema (later sessions)
├── ai/            # cost-cascade router/rules/local_qwen/claude (later sessions)
└── schemas/       # shared response schemas (later sessions)
```
