## 2. アーキテクチャ（確定）

- **Tauri Core (Rust)** が司令塔・**DuckDB/SQLite の単一ライター**。計算（コンポジット/クロス/アラート）、スケジューラ（tokio・ソース別 cadence・指数バックオフ）、IPC、資格情報を担う。
- **Python サイドカー (FastAPI/uv)** はステートレスな取得＋AI。`127.0.0.1` 限定・**セッショントークン認証（401）**。
- **フロント (Next.js 静的エクスポート)** は表示専用。外部 API を直接叩かず Rust 経由（`invoke`＋イベント）。
- **AI コストカスケード**: Tier1 ルール → Tier2 ローカル Qwen(RTX 5090) → Tier3 Claude API（日次予算ゲート）。生成 tier を必ず記録・表示。

## 3. 品質・検証

- ユニットテスト **19件 green**（地合いコンポジット 10・アラート照合 5・クロスマーケット 4／いずれも純関数）。
- 実データ検証: yfinance(米/日指数・VIX/DXY/10年・セクター・ブレッドス)、CFTC COT、為替、Google ニュース＋Tier1 要約、FOMC/日銀カレンダー、クロスマーケット 2 件、アラート発火→履歴→イベントをライブ確認。
- 鮮度: 全データに `ingested_at`/`source_meta`、UI に JST 最終取得時刻と `未取得/エラー` を明示。オフライン時は DuckDB キャッシュで描画。

## 4. 既知の制約 / 資格情報で有効化される機能

`.env`（dev）または設定 UI（キーチェーン）に登録すると自動で有効化:
- **FRED_API_KEY**: 米2年金利・HY OAS・広義ドル（クレジット寄与が点灯、カバレッジ向上）。
- **JQUANTS_REFRESH_TOKEN**（or メール/パスワード）: 投資部門別・信用・空売り・公式 TOPIX（海外勢フロー寄与が点灯）。無料プランは約12週遅延。
- **ANTHROPIC_API_KEY**: ブリーフ/要約の Tier3 昇格（予算ゲート）。
- **EDINET_API_KEY**: 開示取得。
- ローカル LLM 稼働（`LOCAL_LLM_ENDPOINT`）: 要約の Tier2 昇格。

## 5. 実行（dev）

```sh
npm install
cd sidecar && uv sync && cd ..
npm run tauri dev
```
