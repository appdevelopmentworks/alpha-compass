### セキュリティ（§4.3 準拠）
- Rust が起動時に**ランダムなセッショントークン**（uuid×2＝64桁）を生成し、環境変数 `ALPHA_COMPASS_SIDECAR_TOKEN` でサイドカーへ受け渡し。
- サイドカーは全リクエストで `Authorization: Bearer <token>` を要求し、不一致／未提示は **401**（定数時間比較・fail-closed）。スタンドアロン検証で `200 / 401（誤トークン）/ 401（無トークン）` を確認済み。
- サイドカーは **127.0.0.1 のみ** にバインド（動的ポート。今回の実行例ではポート 65378）。
- 資格情報は本セッションでは未導入（平文保存なし）。`.env` は dev 専用・Git 管理外（既存方針どおり）。

### データベース初期化
